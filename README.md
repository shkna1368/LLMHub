# LLMHub

**LLMHub** is a Java-based platform built with Spring Boot and Vaadin that allows seamless integration with multiple Large Language Models (LLMs).

## Supported LLMs
- **Gemeni**
- **OpenRouter** (including OpenAI, Claude, and all models available in OpenRouter)
- **Ollama**
- **GitHub Models**

## Technologies
- Java 17+
- Spring Boot
- Spring AI
- Vaadin (for UI)

## Features
- Flexible integration with multiple LLM providers
- Simple and interactive UI with Vaadin
- Extensible architecture for AI-powered applications
